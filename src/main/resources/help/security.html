Security Feature

The Security Feature displays all the events that might be considered a security risk.

The functionality provided by this Feature is customisable using via the preferences. To get you started the following API calls are monitored by default, though you can remove these if you wish.

  • AuthorizeSecurityGroupEgress
  • AuthorizeSecurityGroupIngress
  • PutGroupPolicy
  • PutRolePolicy
  • PutUserPolicy
