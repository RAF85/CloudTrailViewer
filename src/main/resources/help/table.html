Table View

Table view shows all the event that have been loaded in chronological order. The table view has these features.
  • Filter area. Located at the top of the table you can filter the values of the table by entering a string here
  • Customisable columns. Right clicking on the table header allow you to show/hide the table columns
  • CSV Export: Located to the right of the Fitler are this button allows you to export the values currently displayed in the table.
  • JSON View : Clicking on a row will open a sidebar that will show the original JSON CloudTrail event
  • Chart : The side bar also provides a Charting option when the values in the table can be viewed in a graphic format.
  • Save : Located to the left of the sidebar toolbar is the save button, this allows the visible sidebar area to be save as an image.
