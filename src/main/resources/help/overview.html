Service Overview

The Service Overview feature breaks down the API calls that have been loaded into individual AWS service. Each service box shows the following information.

  • The service name at the top of the box
  • The total number of API Calls that were loaded at the bottom of the box
  • The main section of the box breaks down the API Calls into two areas
    • On the left (the blue colour) are the total number of API calls made by AWS on your behalf while monitoring your resources
    • On the right (the greeny colour) are the total number of API calls made you your IAM User/Group/Roles
    • Under each section is the Peak Events per Second/Minute/Hour

Clicking one of main total values will display the API calls associated with the total.
