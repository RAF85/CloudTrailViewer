GeoData

The GeoData feature breaks down the API calls that have been loaded into the SourceIP address of the API call.

The information is shown based on the city that the IP address is located. This information is provided by a MaxMind database and API that resolve the IP to the appropriate City. The database provided with the application is a free 'Lite' database and is not guaranteed to be 100% accurate.

As with the rest of the application selecting one of the Citys (or IP Address') will show all the events that originated at that location.

If you would like to know where in the world a City or IP Address is location then clicking the Map icon at the top right of the interface will open a web browser displaying the information
